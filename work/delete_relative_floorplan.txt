Name
       delete_relative_floorplan - Removes the relative floorplan information from the database

Syntax
       delete_relative_floorplan [-help] {<obj_name_list>| -all}

Description
       Removes  the  relative  floorplan  information from the database. It deletes all relative floorplan
       constraints, all relative constraints for the specified object(s), or  removes  the  last  executed
       relative  floorplan action from the database. Use this command after running create_relative_floor-
       plan.

       Notes:

              * The arrows showing relative constraints for the specified objects  are  removed  from  the
                floorplan display.

              * The delete_relative_floorplan command accepts an array of objects.


Parameters
       -all   Delete all relative floorplan constraints.

              Data_type: bool, optional

       -help  Outputs a brief description that includes type and default information for each delete_rela-
              tive_floorplan parameter.

              For a detailed description of the command and all of its parameters, use the man command:

              man delete_relative_floorplan

       <obj_name_list>
              Deletes all relative constraints for the specified object(s).

              Note: If you do not specify <obj_name_list>, this command removes the last executed relative
              floorplan action from the database.

              Data_type: string, optional


Example
              * The following command deletes obj1, obj2, and obj3 from the database:

                delete_relative_floorplan {obj1 obj2 obj3}
